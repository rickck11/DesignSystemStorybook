#### How to run:

Step zero, configure the `.env.local` file: 

```env
NEXT_URL=http://localhost:3000

STRIPE_PUBLIC_KEY=your_stripe_public_key
STRIPE_SECRET_KEY=your_stripe_secret_key
```

First, install and run the development server:

```bash
npm install

npm run dev
```
